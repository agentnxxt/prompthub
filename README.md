# PromptHub

A content portal for AI prompts, skills, tools, and agent resources. Built as a Liferay theme + page templates.

## Target Instance

- **URL**: http://51.75.251.56:8080
- **Platform**: Liferay 7.4 CE

## Structure

```
src/
├── templates/       # Freemarker page templates
├── css/             # Theme stylesheets
└── js/              # Frontend JavaScript
content/
├── prompts/         # Prompt library content
├── skills/          # Agent skills catalog
├── tools/           # Tools & integrations
└── agents/          # Agent directory
```

## Deployment

Deploy the theme WAR to Liferay's `/deploy` directory or via the Liferay admin panel.
