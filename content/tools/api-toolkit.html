LLM Gateway (LiteLLM)

PropertyValue
URLhttps://llmgateway.agnxxt.com:4000
ProtocolOpenAI-compatible REST API
AuthBearer token (master key)
ModelsOpenAI, Anthropic, Voyage, and custom
# Chat completion
curl https://llmgateway.agnxxt.com:4000/v1/chat/completions \
  -H "Authorization: Bearer $LITELLM_KEY" \
  -H "Content-Type: application/json" \
  -d '{"model": "gpt-4o", "messages": [{"role": "user", "content": "Hello"}]}'

Translation (LibreTranslate)

PropertyValue
URLhttp://72.61.244.208:5555
Languagesen, es, fr, de, pt, ar, zh, ja, ko, hi, ru
AuthNone required
# Translate text
curl -X POST http://72.61.244.208:5555/translate \
  -H "Content-Type: application/json" \
  -d '{"q": "Hello world", "source": "en", "target": "es"}'

# Response: {"translatedText": "Hola mundo"}

PDF Generation (Gotenberg)

PropertyValue
URLhttp://72.61.244.208:3100
FormatsHTML to PDF, Office to PDF, URL to PDF
AuthNone required
# HTML to PDF
curl -X POST http://72.61.244.208:3100/forms/chromium/convert/html \
  -F "files=@index.html" -o output.pdf

# URL to PDF
curl -X POST http://72.61.244.208:3100/forms/chromium/convert/url \
  -F "url=https://example.com" -o output.pdf
